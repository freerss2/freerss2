"Site to Feed" definition
Site-to-Feed

This technology allows you to define RSS that is never generated by site itself. FreeRSS is able to get pure articles after providing patterns for extraction of article title, link and content. This process could be done in few simple steps: set site origin (URL and encoding), define parsing expression and finally build article template from parsing results.
